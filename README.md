# Arc Multichain AI USDC Agent Controller

> **"Built on top of Arc Multichain Engine, bridging the gap between Web2 AI Prompts and Web3 Smart Contracts."**

An advanced, premium AI-powered decentralized application (dApp) designed to control and route multi-chain USDC transactions seamlessly through natural language commands. 

---

## 🚀 Key Features

* **AI Prompt Processing:** Execute direct multi-chain transfers just by typing simple instructions (e.g., *“Send 2 USDC to 0x...”*).
* **Live Gas Fee Estimator:** Real-time tracking of gas conditions to ensure seamless and optimized execution.
* **One-Click Utilities:** Integrated secure QR Code generator for receiving assets and one-click address copier.
* **Recent Transactions Ledger:** A built-in history ledger tracking real-time successful and failed dashboard transactions.
* **Crystal Clear Vector Interface:** High-performance UI built with Tailwind CSS, supporting crisp modern branding elements.

## 🛠️ Tech Stack & Protocols Used

* **Core Engine:** Arc Unified Balance Engine & Interchain Bridge Protocol
* **Web3 Integration:** Ethers.js (v6)
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6)
* **Libraries:** QRCode.js for secure dynamic data rendering
